TITULO DO PROJETO: [Exercicios] Encapsulamento pacotes e relacionamentos

OBJETIVO DO PROJETO:
1. Faça um programa para controle de empréstimo de livros, com as classes Emprestimo, Livro e Pessoa.
2. Faça um programa de agenda telefônica, com as classes Agenda e Contato.
3. Faça um programa para o Mecanico que possua as classes veiculo, proprietário e concertos.
4. Crie um sistema para controlar filmes em um software como o Netflix. Classes Filme, Serie, MinhaLista e Lançamentos.

ATENÇÃO:
- Cada exercício deverá estar em um pacote.
- Escreva de 3 a 4 atributos por classe.
- Utilize ArrayList no exercício 4
- O padrão camelCase deverá ser respeitado.
- O projeto final deverá estar em formato zip e enviado em um único arquivo.
